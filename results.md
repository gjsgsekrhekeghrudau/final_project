# Сервис для подготовки к собеседованиям через LLM: результаты работы

### Елизавета

Спроектировала и разработала интуитивно понятный веб-интерфейс.
Создала надежную, масштабируемую и безопасную серверную архитектуру, которая обеспечивает стабильную и контекстно-зависимую беседу с ИИ-интервьюером. Создала docker файл.
Сформулировала и описала функциональные требования к системе для Льва, включая логику диалога, условия показа подсказок и параметры оценки.
Провела тестирование MVP, собрала обратную связь.

### Лев

Разработан backend-сервис на FastAPI для обработки диалогов пользователя с LLM.
Реализован единый эндпоинт /api/chat, принимающий историю сообщений и режим работы сервиса.
Ответы LLM возвращаются в унифицированном формате, пригодном для дальнейшего расширения (метаданные, оценка ответа, баллы).
Разработал прототипы интерфейса, на основе которых реализовывался фронтенд. Определил оптимальное расположение элементов для снижения когнитивной нагрузки.
Реализовал динамический интерфейс чата с отображением ролей (Пользователь / Интервьюер).
Создал интерактивную систему подсказок: разработал методы для плавного доступа к подсказкам и эталонным ответам без прерывания потока собеседования.
Обеспечил безопасность: внедрил безопасное хранение ключей API.
Достижение: Создал фокусную, минималистичную и снижающую тревожность среду, которая позволяет пользователю полностью сосредоточиться на процессе собеседования.

### Андрей

Разработал серверное приложение, которое выступает в роли посредника между фронтендом и ChatGPT API.
Отправка сообщения пользователя и получения ответа от LLM.
Управления контекстными подсказками.
Реализовал эффективную систему промптов: разработал шаблоны, которые делают из LLM профессионального интервьюера, и механизм управления контекстом диалога.
